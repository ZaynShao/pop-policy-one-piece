## 5. 落地建议(给 V0.3)

1. **Step 1** · 顶栏 + 路由骨架(`<AppLayout>` shell + 5 角色登录后默认 path)

4. **Step 4** · 双大盘地图画布(高德地图 SDK 接入 + 左/右抽屉空壳)
5. **Step 5** · 数据接入(回到 β/γ 业务实体)
